    Detailed Description

    This class acts as the main window for the common print dialog. It is a container to widget objects namely tabs, root, preview and controls.

    masterLayout |-tabs |-root -- preview |-controls

    Member Function Documentation

    QCPDialog::QCPDialog(QPrinter * printer, QWidget * parent = Q_NULLPTR)

    Constructs QCPDialog with parent as the parent widget and printer as the printer. The printer handles all the job options that get reflected in the preview as well as the final job output.

    void QCPDialog::addMaximumPrintCopies(char * _copies)

    This function takes the range _copies for the number of copies and sets the maximum value allowed in the dialog.

    void QCPDialog::addPagesPerSize(char * pages)

    void QCPDialog::addPrinter(char * printer, char * backend)

    Whenever a new printer with name printer and backend backend is discovered in the backends, this function is called which adds that printer to the UI.

    [slot] void QCPDialog::cancelButtonClicked()

    This function acts as a slot for the signal cancelButtonClicked emitted from the Controls.qml file. The signal is emitted when the user clicks on the "Cancel" Button in the sidebar.

    void QCPDialog::clearPaperSizeModel()

    This function clears the model paperSizeModel which holds a list of printer-supported media options.

    void QCPDialog::clearPrinters()

    This function clears the list of printers shown in the UI.

    void QCPDialog::clearStartJobsModel()

    This function clears the model startJobModel which holds a list of printer-supported job hold options.

    int QCPDialog::exec()

    QString QCPDialog::information()

    Returns the information regarding the dialog.

    void QCPDialog::init_backend()

    This function is called whenever a new instance of the dialog is created. This function helps create a new FrontendObj for a new instance of the dialog and connects the FrontendObj to the dbus for further communication.

    [slot] void QCPDialog::newPrinterSelected(const QString & printer)

    Whenever a new printer with name printer is selected by the User, this function updates all the options shown in the dialog with the options and values supported by the selected printer.

    [slot] void QCPDialog::remotePrintersToggled(const QString & enabled)

    When enabled is set to "true", the dialog shows all printers including remote printers whereas when it is not set to "true", the dialog hides all the remote printers and shows only the local printers.

    [slot] void QCPDialog::swipeViewIndexChanged(qint32 index)

    This function acts as slot for the signal swipeViewIndexChanged which is emitted from Root.qml It sets the property "index" in Root.qml to index

    [slot] void QCPDialog::tabBarIndexChanged(qint32 index)

    This function acts as slot for the signal tabBarIndexChanged which is emitted from Tabs.qml It sets the property "index" in Tabs.qml to index

    void QCPDialog::updatePaperSizeModel(char * media, int isDefault)

    Adds a new paper defined by media to the existing model paperSizeModel. The isDefault parameter checks if the given media is to be set as the default for the printer.

    void QCPDialog::updateStartJobsModel(char * startJobOption)

    Many printers come with a hold option which specifies when the job should start after a user clicked the "Print" button in the UI. This function adds a new startJobOption to the existing model.
